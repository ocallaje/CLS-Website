Campbell Lab Services Portal

This webpage acts as a central hub to connect all apps, software and digital content. Your TCD credentials will provide access where asked for a username and password. Internal services are only available on college networks meaning you must be plugged into a port, use the staff wifi (TCDwirless) or use the TCD VPN off campus. External services are available from anywhere.

Our Cloud Services

Synology Drive

Data storage platform. Access here via web or via PC filesystem. Personal and group storage available. Data can be synced across devices.

SeqWorks

RNAseq pipelines application. Download the latest version from github to use direct from your device.

Shiny web apps

Frontend applications created and hosted in R Shiny via "shiny server". Current apps = GBM_Inventory.

elabftw

Open source self-hosted electronic lab notebook. Learn more at https://www.elabftw.net/

Standalone Software

A collection of useful software n readily available online, inlcuding FOVAS, lab instrument software and other useful tools.

Synology apps

A collection of both desktop and mobile apps supported by our infrastructure. Drive, Chat, Notes and more!

Lab Management

Lab Documentation

This documentation portal records procedures and protocols for a range of lab- and college-wide practices from ordering consumables to interacting with various Campbell-Lab-Services. These docs server as training and reference guides, and can be updated as often as required.
