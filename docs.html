Add or Remove Users to CLS

How to give a new lab member access to Campbell Lab Services or remove old users

Domain Group

What is a domain group? A domain group is a group of usernames that has access to specific servers or resources. The domain group set up for the lab is called NAS-Campbell-Lab. Access to CLS is granted for users on this list.

How to Update the User List

The ability to update the user list is controlled by TCD IT Services, so we can not directly edit this ourselves. A request needs to be sent to IT Services who will update the for us. IT requests can be sent via ask.tcd.ie.

Once logged in you can click "ask a question" and then click "Log Enquiry".

Select IT Services Issue as the category so it gets to the correct department. You can use the following template to fill out the message box

Subject: Domain Group Update

Message: Hi,
Could I update the domain group called "NAS-Campbell-Lab" please?
I want to add
jhenders
amcglinc

and remove
hardfordd

Thanks very much!
"your name here"

IT will respond in a day or so and then the new user will be able to log in!

Reset Password

The following section details how to reset your TCD password and renew access to TCD and CLS services

New Password

TCD requires changing your password every 180 days for security reasons. You will receive an email when your password is due to be changed. To do so, you can visit the following link at password.tcd.ie. This will allow you to:

  1. Change your password if you know your current password or if it has expired
  2. Reset your password if you forget it
  3. Unlock your account

Further details can be found on the TCD website here.

CLS Access

Your TCD password governs access to Campbell Lab Services including data storage, apps and other cloud services. When you change your TCD password you will be able to use this new password for everything including CLS going forward. There shouldn't be a waiting period between changing your password and using any CLS services.

You will however have to manually update your password if you have automatic connections to the server either via windows or the drive client app. Read on for details

Update Password on Windows

If you access server storage locations from your PC using a "network location", you will have to update your windows settings with your new password. A network location looks like this:

To change your password for all of these types of folders, do the following:

  1. Click "Start"
  2. Type "Credential Manager" and open the program
  3. Click on Windows Credentials
  4. Click on the credential you want to change. For example, the Campbell Lab Server could be listed as either gen003024 or 134.226.153.2.
  5. Click on Edit
  6. Enter your new password in the password field
  7. Repeat for any other server connections you may have

Update Password on Drive Client

If you access server storage locations from your PC using Synology Drive Client, you will have to update your connection settings with your new password. Drive client connections look like this:

To change your password for this type of connection, do the following:

  1. Click "Start"
  2. Type "Synology Drive Client" and open the program
  3. Click on the server connection you want to edit e.g. Campbell Lab. (click the server icon. not any subfolder icons)
  4. Click Edit Connection at the top.
  5. Enter your new password in the password field
  6. Click OK
  7. Repeat for any other server connections you may have

Password Lockout

Occasionally when a password is changed, your account can get locked. This can happen if you try to access a service using the wrong password too many times. For us, this is usually if you have a server connection that is trying to connect using your old password. To fix this issue, manually update your new password as described above for Windows or Drive Client connections. Then, unlock your account by visiting password.tcd.ie, enter your username and click Unlock My Account.

Note: you will have to set up recovery questions if you have not done so already to access this feature.

Ordering System

Ordering Guidelines

If you notice any item is low in stock

  1. Write it on the whiteboard on the freezer in the main lab.
  2. Put it in the science warehouse basket or ask for a quote
  3. Put the details on the excel sheet

If ordering general consumables (plastic wear or anything on the stock check sheet on fridge):

  1. Open the Order sheet Excel file found in the MC Group Share Dropbox or on the server
  2. Search for the last time that item has been order and find the catalogue number
  3. Enter this number into science warehouse and add the item to the basket with the required quantity.
  4. Return to the order sheet and input the details (name, supplier, cat code, price per item, quantity)

Science Warehouse

Log in to sciware here Science Warehouse and use your personal account to log in

Processing a Science Warehouse Basket

  1. Ask for the grant and the code to be charged.
  2. Open Science warehouse basket and add current basket to a saved basket
  3. Create a new saved basket and put your name, lab and the grant code in the name section, e.g. Avril-MC-123456
  4. Go to the saved basket, check items are correct then press "share basket" and add
    1. Sullivan, Rachel
    2. Campbell, Philip
    3. Campbell, Brenda
  5. Make sure all items have been added to the excel sheet (this cannot be done later as once processed, the saved basket disappears).
  6. Put your name, the date and the grant used in the excel sheet.
  7. In science warehouse, remove the items from the general basket (not the saved basket) to avoid them being ordered again
  8. When the order is confirmed, you may receive an email from the company with the PO number. Enter this into the excel sheet beside the other details.
  9. When the order is delivered, check storage instructions on the product information sheet (found with order or online) and put away. Add in the date the item was delivered to the excel sheet.

Non-Catalogue Requests

If an item is not on science warehouse, you will need to ask for a quote. This will generally apply for rarely ordered items such as AquaPolymount or Abcam antibodies or maybe for specialist items you need for your project.

  1. Email the rep for the company and give them the catalogue code, name and quantity of the item you want to order and ask for a quote. In this email, also ask if there is a delivery fee and if they can provide a discount (sometimes will for Trinity etc).
  2. When they respond, you fill out a non-cat form with the quote number, grant code, item name, quantity and price and delivery fee.
  3. Send the original quote and the non-cat form to genorder@tcd.ie and ask them to generate a PO with the grant code you give them. It is a good idea to ask them (ie Philip or Rachel usually) to send you the PO when complete so you can add this to the excel sheet as well (fill out the same as above).
